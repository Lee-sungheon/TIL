# branch
> git branch 명령어 정리

## 개념
- 나무가지라는 의미로 코드의 분기를 의미함

## 확인
 - `git branch`

 ## 생성
 - `git branch <new branch name>`

 ## 이동
 - `git switch <branch>`

 ## 삭제
 - `git branch -d <branch name>`

## 합병
- `git merge <branch name>`
