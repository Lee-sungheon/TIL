## React Native

#### React Native

- Facebook에서 만든 오픈소스 프로젝트
- 2015년 3월
- 기존 하이브리드앱에서 겪던 성능문제 없이 JavaScript를 이용하여 ios, android 개발가능

- 장점
  - 쉬운 접근성 (JavaScript)
  - 비용 절감
  - Fast Refresh
- 단점
  - 네이티브 앱과의 성능 차이
  - 디버깅
- Bridge : js와 native 사이에서 통신을 담당해주는 역할



#### 프로젝트 생성

- Expo

  ```bash
  npm install -g expo-cli
  expo login
  expo init my-first-expo
  expo start
  ```
